Objects

Class-based selectors which define undecorated design patterns, for example media object known from OOCSS.

Format: .c-object-name[<element>|<modifier>] {}

.o-object {}
.c-object__child {}
.c-object--modifier {}


/* Example */

.o-page {}
.o-page__card {}
.c-page--center {}

<div class="o-page o-page--center">
    <div class="o-page__card"></div>
</div>
