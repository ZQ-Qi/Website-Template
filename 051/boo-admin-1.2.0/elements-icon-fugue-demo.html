Fugue Icon fugue sprite Boo

Boo Fugue Sprite CSS and image provides easy access to 3,300 Fugue icons. The current version of the icon of Administrators Boo is 3.3.

Just use a prefix to the name icons .fugue- : address-book - .fugue-address-book html <i class="fugue-address-book"></i> result

The Fugue icon set is released under the Creative Commons Attribution 3.0 license, so if you use it, be sure to provide a link back to p.yusukekamiyamane giving them credit for the icons.
