Teal Palette Classes

Teal color palette contains 13 colors: 1 main teal color and 4 lighten colors, 4 darken colors and 4 Accent colors. All colors works perfectly with any bootstrap components, form components or any stack controls, that makes Stack very effective, flexible and configurable.

Please note: You can use .bg-teal and .btn-teal as main colors, but if you want to use lighten, darken or accent colors, use bg-* bg-lighten-NUMBER for lighten color, bg-* bg-darken-NUMBER for darken color and bg-* bg-accent-NUMBER for accent colors.

Class Description
.bg-teal Teal Class for background color.
.bg-teal
.bg-lighten-*		 For lighten teal background color, this two classes needed. Here *: 1,2,3,4 lighten teal color options.
.bg-teal
.bg-darken-*		 For darken teal background color, this two classes needed. Here *: 1,2,3,4 darken teal color options.
.bg-teal
.bg-accent-*		 For accent teal background color, this two classes needed. Here *: 1,2,3,4 accent teal color options.
