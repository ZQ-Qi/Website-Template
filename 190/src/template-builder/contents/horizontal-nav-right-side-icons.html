This table contains class which can be used in horizontal navigation to hide dropdown arrow.

Classes Description
.navbar-icon-right To set horizontal navigation icon position to right, you need to add .navbar-icon-right class in horizontal navigation <div> tag. Refer HTML markup line no 12.

HTML Markup

This section contains HTML Markup to create light vertical navigation. This markup define where to add css classes to make vertical navigation light.

PUG Configuration

Modern Admin use PUG as template engine to generate pages and whole template quickly using node js, for getting start with PUG usage & template generating process please refer template documentation.

PUG Variables

This table contains required PUG variables to generate horizontal navigation with right side icons.

Block Variable Datatype Value Description
horizontalMenuConfig menuIconRight boolean true horizontalMenuConfig block has a menuIconRight variable, it contains horizontal navigation icon specific settings in that you need to set true.
PUG Code

Use following PUG code to generate page or template having vertical light navigation.

  • Line no 20-21: Set horizontal navigation menu icon right setings to menuIconRight variable value as true in horizontalMenuConfig block, it contains horizontal navigation icon specific classes.
  • Line no 27-28: content block has content section html file included include ../contents/horizontal-navigation-right-side-icons.html, which can be customizable on page level.

If you want to use this options on page level you need to define this configuration option on PUG page it self. This template has one example pug file for horizontal-navigation-submenu-click.pug, however you can use it on template level but it will generate template all pages with the same configuration option.

                    
                    block pageVars
                        - var pageTitle    = "Right Side Icon"
                        - var pageSubTitle = "Horizontal navigation right side icon"
                        - var description  = "This option change top level horizontal navigation menu item icon position to right side."
                        - var activeMenu   = "horizontal-nav-right-side-icons"

                    extends template

                    append horizontalMenuConfig
                        - var menuIconRight = true

                    append breadcrumbs
                        +breadcrumbs([{url:"index.html",name:"Home"},{url:"#",name:"Horizontal Navigation"}, {name:"Right Side Icon"}])

                    //- Include page content in page block
                    append content
                        include ../contents/horizontal-nav-right-side-icons.html

                    //- Vendor CSS
                    //------------------------------
                    //- Add Vendor specific CSS
                    append vendorcss

                    //- Page CSS
                    //------------------------------
                    //- Add custom page specific CSS
                    append pagecss

                    //- Vendor JS
                    //------------------------------
                    //- Add vendor specific JS
                    append vendorjs

                    //- Page JS
                    //------------------------------
                    //- Add custom page specific JS
                    append pagejs
