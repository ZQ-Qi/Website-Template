Marked Text

For highlighting a run of text due to its relevance in another context, use the <mark> tag.

Donec ac ipsum luctus, hendrerit est vitae, convallis tortor.

Deleted Text

For indicating blocks of text that have been deleted use the <del> tag.

Nulla eleifend quis felis et sollicitudin. Suspendisse ut nisl leo. Donec vitae libero accumsan, interdum leo.

Underlined Text

To underline text use the <u> tag.

Nullam euismod odio enim. Vestibulum hendrerit enim eget tellus tempus ultrices.

Inserted Text

For indicating additions to the document use the <ins> tag.

Nam maximus mattis neque, at pharetra arcu blandit eget. Morbi arcu nisl.

Strikethrough Text

For indicating blocks of text that are no longer relevant use the <s> tag.

Blandit justo diam sit amet diam. Pellentesque pharetra consequat massa.

Bold Text

For de-emphasizing inline or blocks of text, use the <small> tag to set text at 85% the size of the parent.

Nam maximus mattis neque, at pharetra arcu blandit eget. Morbi arcu nisl.
