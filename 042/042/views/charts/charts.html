Angular version of chartist.js created with one line of html. Simply add <chartist> element in your html with full customization attributes like type, data, opts, res-opts etc. This version comes with pre defined hover tooltip so you don't have to write one. Just set attribute tip to 'true' in <chartist> element. You'll find here most of the chart options built with angular chartist version included in the template.
