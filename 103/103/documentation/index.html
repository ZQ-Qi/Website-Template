Minoral 1.3 - Responsive admin theme

Responsive admin theme builded on Twitter Bootstrap v3, with most of mobile devices support.

  • created: 02/13/2014
  • latest update: 14/5/2014
  • by: Tattek
  • www.tattek.com/minoral

Thank you for purchasing my theme. If you have any questions that are beyond the scope of this help file, please feel free to email via my user page contact form here. Thank you so much!

Features & Example pages

Features:

 

  • Responsive design
  • SCSS
  • Font Awesome
  • Integrated charts (jQuery flot, Sparklines, JustGage, Rickshaw, Morris.js, EasyPieChart)
  • Dynamic tables (Datatables)
  • Growl style notifications (using jGrowl)
  • Calendar (using Fullcalendar)
  • Form wizard (using bootstrapWizard)
  • Validated HTML5
  • Sliders, Togglers
  • WYSIWYG editor (using Summernote)
  • Form validation (using parsley.js)
  • Color Schemes
  • Multiple selectboxes (using Chosen)
  • Easy customized tiles
  • Show snippet on almost every element

HTML Structure

Preloader:

First element you can find in body is Preloader, preloader is showing loading progress when page is loading, after load it dissapear.

<!-- Preloader -->
<div class="mask"><div id="loader"></div></div>
<!--/Preloader -->

Wrap & Fluid:

After Preloader there are going two divs first wrap whole page and another is making it fluid

<!-- Wrap all page content here -->
<div id="wrap">    

   <!-- Make page fluid -->
   <div class="row">
      ...
   </div>
   <!-- /Make page fluid -->

</div>
<!-- /Wrap all page content here -->

Navbar:

Navbar contain some important parts like as: Branding, Quick Actions (tasks, reminders), Search, User Controls (messages, control panel) and Sidebar. Quick actions, search and user controls are collapsing to the sidebar if window width is less than 768px.

<!-- Fixed navbar -->
<div class="navbar navbar-default navbar-fixed-top" role="navigation">       

   <!-- Branding -->
   <div class="navbar-header col-md-2">
      <a class="navbar-brand" href="index.html">
         <strong>MIN</strong> <span class="divider vertical"></span> ORAL
      </a>
     
      <div class="sidebar-collapse">
         <a href="#">
            <i class="fa fa-bars"></i>
         </a>
      </div>
   </div>
   <!-- Branding end -->

   <!-- .nav-collapse -->
   <div class="navbar-collapse">

      <!-- Content collapsing at 768px to sidebar -->
      <div class="collapsing-content">

         <!-- Quick Actions -->
         <ul class="nav navbar-nav">
               
            <li class="divided">
               <a href="#" class="page-refresh"><i class="fa fa-refresh"></i></a>
               <span class="divider vertical"></span>
            </li>

            <li class="dropdown quick-action tasks">
               ...
            </li>

             <li class="dropdown quick-action notifications">
               ...
             </li>

           </ul>
           <!-- Quick Actions end -->

           <!-- Search -->
           <div class="search">
             <input type="text" placeholder="&#61442; Search in form wizard...">
           </div>
           <!-- Search end -->

           <!-- User Controls -->
           <div class="user-controls">
             <ul>

               <li class="dropdown messages">
                 ...
               <li>

               <li class="dropdown settings">
                 ...
               </li>

             </ul>
           </div>
           <!-- User Controls end -->

      </div>
      <!-- /Content collapsing at 768px to sidebar -->


      <!-- Sidebar -->
      <ul class="nav navbar-nav side-nav" id="navigation">

         <li class="collapsed-content">
            <!-- Collapsed content pasting here at 768px -->
         </li>

         <li class="user-status status-online" id="user-status">
           ...
         </li>

         <li class="dropdown">
           ...
         </li>

         <li class="dropdown open">
           ...
         </li>

       </ul>
       <!-- Sidebar end -->

     </div>
     <!--/.nav-collapse -->

 </div>
 <!-- Fixed navbar end -->

Content:

In page content you can find Breadcrumbs, Submenu and finally the main content in div with class .main

<!-- Page content -->
   <div id="content" class="col-md-12">

      <!-- breadcrumbs -->
      <div class="breadcrumbs">

         <ol class="breadcrumb">
            <li><a href="index.html"><i class="fa fa-home"></i> Home</a></li>
            <li><a href="index.html">Dashboard</a></li>
            <li class="active">Overview</li>
         </ol>

      </div>
      <!-- /breadcrumbs -->

      <!-- submenu -->
      <div class="submenu">
         <h1>Dashboard</h1>
         <div class="collapsed">
            <a href="#" id="submenutoggle"><h1>Dashboard <i class="fa fa-bars"></i></h1></a>
         </div>
         <ul class="nav nav-tabs">
            <li class="active"><a href="index.html">Overview</a></li>
            <li><a href="mail.html">Inbox</a></li>
         </ul>
      </div>
      <!-- /submenu -->

      <!-- content main container -->
      <div class="main">
         CONTENT IS HERE
      </div>
      <!-- /content main container -->

</div>
<!-- /Page content -->
Most of the content is placed in tiles, every tile can have it's own header, widget, body, right side, left side and footer. 
 
<!-- tile -->
<section class="tile left-nav">
   
   <!-- tile header -->          
   <div class="tile-header">           
      TILE HEADER IS HERE
   </div>
   <!-- /tile header -->

   <!-- tile widget -->
   <div class="tile-widget">         
      TILE WIDGET IS HERE     
   </div>
   <!-- /tile widget -->

   <!-- tile left-side -->
   <div class="tile-left-side">                 
      TILE LEFT SIDE IS HERE
   </div>
   <!-- /tile left-side -->

   <!-- tile body -->
   <div class="tile-body">
      TILE BODY IS HERE   
   </div>
   <!-- /tile body -->

   <!-- tile right-side -->
   <div class="tile-right-side">           
      TILE RIGHT SIDE IS HERE
   </div>
   <!-- /tile right-side -->

   <!-- tile footer -->
   <div class="tile-footer">                 
      TILE FOOTER IS HERE
   </div>
   <!-- /tile footer -->

</div>
<!-- /tile -->
Every part of tile can have it's own color option by adding class .color and color defining class such as .red, same way you can colorize whole tile.
 
Every tile in demo page have view code button, so more documentation about almost every element you can find there.

SCSS:

SCSS files you can find in root/css/scss folder:

minoral.scss // main scss file
_schemes.scss // color schemes file
_mixins.scss // mixins file
_colors.scss // colors file

SCSS structure:

Every section is divided with description like this:

/*********************************/
/************* FORMS *************/
/*********************************/

At the end of this file are settings for responsiveness

/**************************************/
/************* RESPONSIVE *************/
/**************************************/

Everything else is default scss

Javascript

Important Javascript files:

1. Most important is jQuery - jQuery is a Javascript library that greatly reduces the amount of code that you must write.

2. Another one is bootstrap.js - default js file for bootstrap

3. Main theme js file is minoral.js - there are some major scripts for theme which are necessary for all pages

4. Every page have it's own <script></script> section where you can find all other functions

5. At the end there are many  plugins which are doing more complex work and the list of them with links to documentation you can find in last chapter of documentation "Sources and Credits".
