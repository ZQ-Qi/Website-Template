Page

Used to strucutre your page layout into sidebar and main content.

Usage

Use .o-page to strucutre your page layout into sidebar and main content.

Don’t forget to add .js-page-sidebar class for hiding / showing sidebar based on the current screen size.

<body class="o-page">
	
    <!-- Notice `.js-page-sidebar` used for toggling sidebar via javascript -->
    <aside class="o-page__sidebar js-page-sidebar"></aside>

    <main class="o-page__content"></main>
</body>
