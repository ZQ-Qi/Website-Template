Documentation

You can pass plugin options using data-options attribute. Please make sure to use valid json code.

Plugin Docs JSON Formatter

Code Example

<input type="text" class="range-slider form-control"
   data-options='{
   "type": "double",
   "grid": true,
   "min": -1000,
   "max": 1000,
   "from": -500,
   "to": 500,
   "step": 250
   }'/>
