Show / hide Datatable

This example shows how FixedHeader can be enabled and disabled using the fixedHeader.adjust() API method. This is can be particularly useful if FixedHeader is used for DataTables that are enclosed in tabs, so the header is disabled when the tab containing it is not visible, and also to then realign the FixedHeader when the table is then made visible again.
