Description

Change navigation menu position to right with the help of flipped navigation menu options. RTL layout has default flipped navigation menu.

Experience it!

This page contain flipped navigation menu options example, check at the left hand side of the page.

CSS Classes

This table contains all classes related to set vertical flipped navigation option.

All these options can be set via following classes:

Classes Description
.menu-flipped To create vertical flipped navigation, you need to add .menu-flipped class in navigation <body> tag. Refer HTML markup line no 4.

HTML Markup

This section contains HTML Markup to create vertical flipped navigation. It define where to add css classes to make vertical flipped navigation.

  • Line no 4: Contain the .menu-flipped class to set vertical navigation flipped.

PUG Configuration

Modern Admin use PUG as template engine to generate pages and whole template quickly using node js, for getting start with PUG usage & template generating process please refer template documentation.

PUG Variables

This table contains required PUG variables to generate vertical flipped navigation.

Block Variable Datatype Value Description
pageConfig menuFlipped boolean true pageConfig block has a menuFlipped variable, it contains navigation specific classes in that you need to set true for flipped navigation.
Modern Admin Template documentation has full list of all blocks, datatype, variables and their description table. Please review it for more configuration options.
PUG Code

Use following PUG code to generate page or template having vertical flipped navigation.

  • Line no 20-21: Set vertical flipped navigation class to menuFlipped variable value as true in pageConfig block.
  • Line no 27-28: content block has content section html file included include ../contents/vertical-flipped-navigation.html, which can be customizable on page level.

If you want to use vertical flipped navigation options on page level you need to define it on page it self. This template has one example pug file for vertical-flipped-navigation.pug, however you can use it on template level but it will generate whole template have vertical flipped navigation.

                
                block pageVars
                    - var pageTitle    = "Flipped Navigation"
                    - var pageSubTitle = "Vertical flipped navigation menu"
                    - var description  = "Flipped navigation menu change the navigation position to right side."
                    - var activeMenu   = "vertical-nav-flipped"

                extends template

                append pageConfig
                    - var menuFlipped   = true

                append breadcrumbs
                    +breadcrumbs([{url:"index.html",name:"Home"},{url:"#",name:"Navigation"}, {name:"Flipped Navigation"}])

                //- Include page content in page block
                append content
                    include ../contents/vertical-nav-flipped.html

                //- Vendor CSS
                //------------------------------
                //- Add Vendor specific CSS
                append vendorcss
                    link(rel='stylesheet', type='text/css', href='../../../app-assets/vendors/css/ui/prism.min.css')

                //- Page CSS
                //------------------------------
                //- Add custom page specific CSS
                append pagecss

                //- Vendor JS
                //------------------------------
                //- Add vendor specific JS
                append vendorjs
                    script(type='text/javascript' ,src='../../../app-assets/vendors/js/ui/prism.min.js')

                //- Page JS
                //------------------------------
                //- Add custom page specific JS
                append pagejs
