Description

Default dropdown menu opens on click. Via data attributes data-open="click" or JavaScript, the dropdown plugin toggles hidden content (dropdown menus) by toggling the .open class on the parent list item.

Experience it!

Please check horizontal layout menu open on click example to experience it live.

HTML Markup

This section contains HTML Markup to create light vertical navigation. This markup define where to add css classes to make vertical navigation light.

  • Line no 4: Contain the data-open attribute has click in body tage. Option data-open="click" will open all the dropdown on click.
        
            <!DOCTYPE html>
              <html lang="en">
                <head></head>
                <body data-open="click" data-menu="vertical-menu" class="vertical-layout vertical-menu 2-column menu-expanded">

                  <!-- fixed-top-->
                  <nav role="navigation" class="header-navbar navbar-expand-sm navbar navbar-with-menu fixed-top navbar-dark navbar-shadow navbar-border">
                      ...
                  </nav>

                  <!-- BEGIN Navigation-->
                  <div class="main-menu menu-light menu-shadow">
                      ...
                  </div>
                  <!-- END Navigation-->

                  <!-- BEGIN Content-->
                  <div class="content app-content">
                      <div class="content-wrapper">
                          <!-- content header-->
                          <div class="content-header row">
                              ...
                          </div>
                          <!-- content header-->

                          <!-- content body-->
                          <div class="content-body">
                              ...
                          </div>
                          <!-- content body-->

                      </div>
                  </div>
                  <!-- END Content-->

                  <!-- START FOOTER DARK-->
                  <footer class="footer footer-dark">
                      ...
                  </footer>
                  <!-- START FOOTER DARK-->

                </body>
              </html>

PUG Configuration

Modern Admin use PUG as template engine to generate pages and whole template quickly using node js, for getting start with PUG usage & template generating process please refer template documentation.

PUG Variables

This table contains required PUG variables to generate vertical navigation light.

Block Variable Datatype Value Description
pageConfig menuOpen string 'click' pageConfig block has a menuOpen variable, it contains navigation specific classes in that you need to set 'click' to open dropdown on click.
Modern Admin Template documentation has full list of all blocks, datatype, variables and their description table. Please review it for more configuration options.
PUG Code

Use following PUG code to generate page or template having vertical light navigation.

  • Line no 20-21: Set dropdown open on click settings to menuOpen variable value as 'click' in pageConfig block.
  • Line no 27-28: content block has content section html file included include ../contents/horizontal-navigation-submenu-click.html, which can be customizable on page level.

If you want to use dropdown menu open options on page level you need to define this configuration option on PUG page it self. This template has one example pug file for horizontal-navigation-submenu-click.pug, however you can use it on template level but it will generate template all pages with the same configuration option.

              
                block pageVars
                  - var pageTitle    = "Sub Menu on Click"
                  - var pageSubTitle = "Horizontal navigation submenu click"
                  - var description  = "Configuration options to open horizontal navigation dropdown menus & submenus on click."
                  - var activeMenu   = "horizontal-nav-submenu-click"

                extends template

                append pageConfig
                  - var menuOpen = "click"

                append breadcrumbs
                  +breadcrumbs([{url:"index.html",name:"Home"},{url:"#",name:"Horizontal Navigation"}, {name:"Sub Menu on Click"}])

                //- Include page content in page block
                append content
                  include ../contents/horizontal-nav-submenu-click.html

                //- Vendor CSS
                //------------------------------
                //- Add Vendor specific CSS
                append vendorcss

                //- Page CSS
                //------------------------------
                //- Add custom page specific CSS
                append pagecss

                //- Vendor JS
                //------------------------------
                //- Add vendor specific JS
                append vendorjs

                //- Page JS
                //------------------------------
                //- Add custom page specific JS
                append pagejs
