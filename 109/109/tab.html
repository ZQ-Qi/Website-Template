Tab 组件说明文档

参数
属性名称 类型 默认值 描述
elem string undefined

内容容器. 1/id选择器 : #id 2/类选择器：.class
closed boolean true

关闭
autoRefresh boolean false

自动刷新
contextMenu boolean false

右键菜单
方法
名称 参数说明 描述
exists

参数名：title

类型：string

检索tab选择卡是否存在，如果存在则返回索引值，不存在返回-1

示例

var index = tab.exists('用户中心');
//e.g. 
//result:1

getTabId

参数名：title

类型：string

获取tab选项卡id

示例

var id = tab.getTabId('用户中心');		
//e.g. 
//result:1488434851738

tabAdd

参数名：data

类型：object

参数说明

data.href //需要加载的地址

data.icon //图标

data.title //标题


添加一个tab选择卡，如果tab选择卡存在则获取焦点

示例

var id = tab.tabAdd({
	href:'#',
	icon:'fa-user',
	title:'百度'
});
